Available commands:

Commands are one per line, and must be one of the following
template <template name> instantiate a template (e.g. StarBeast)
import <subtemplate>? file (arg1, arg2, ...)? import data using template <subtemplate> if non-standard template is required
sub (idpattern =)? <subtemplate>(key1=value1,key2=value2,...)instantiate a sub-template
set idpattern '=' STRING set primitive value
use (inputidentifier '=')? <subtemplate> (key1=value1,key2=value2,...)?use subtemplate (at particular input)
add <subtemplate> (value1, value2, ...) add subtemplate
rm idpatternremove input
taxonset name = taxon1 taxon2 ... specify taxonset with specific name
link (clock|site|tree) <partition>? link selected partitions
unlink (clock|site|tree) <partition>? unlink selected partitions
save <filename> save current model to file
? or help show help message
quit or end exit program
