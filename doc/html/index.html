Available commands:

Commands are one per line, and must be one of the following
template <template name> instantiate a template (e.g. StarBeast)
import <subtemplate>? file import data using template <<ubtemplate> if non-standard template is required
(idpattern =)? sub <subtemplate>(key1=value1,key2=value2,...)
partition partitionpattern
link (clock|site|tree) link selected partitions
unlink (clock|site|tree) unlink selected partitions
? or help show help message
save <filename> save file
quit or end exit program
